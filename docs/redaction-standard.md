# Redaction Standard

Redact tenant IDs, domains, UPNs, email addresses, object IDs, app IDs, client IDs, client secrets, certificate thumbprints, tokens, admin names, internal app names, group names, and exploitable findings.

Use placeholders like `REDACTED_TENANT_ID`, `example.redacted`, `user001@example.redacted`, `Application 001`, `Group 001`, and `REDACTED_GUID`.
